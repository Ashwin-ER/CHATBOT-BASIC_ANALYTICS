# Flask Chatbot Application

This is a simple Flask-based chatbot application that allows users to submit queries and provide satisfaction ratings. The application tracks user interactions and provides an analytics dashboard to display key metrics about chatbot usage.

## Features

- User-friendly chatbot interface for submitting queries.
- Satisfaction rating system (1-5) for user feedback.
- Analytics dashboard displaying:
  - Total number of queries.
  - Average user satisfaction rating.
  - Most common topics queried.
  - History of all user interactions.
